# **App Name**: RahulList

## Core Features:

- Progress Tracking: Update and track episode/chapter progress for anime and manga.
- Custom Lists: Organize titles into custom lists such as 'Watching', 'Planned', and 'Completed'.
- Title Search: Search for new anime and manga titles using a comprehensive database.
- Dashboard Summary: View a personalized dashboard with statistics on your anime and manga consumption.

## Style Guidelines:

- Dark background (#121212) to enhance visual appeal and reduce eye strain.
- Primary color: Vibrant purple (#A78BFA) to create a modern and engaging look.
- Accent color: Electric blue (#38BDF8) for interactive elements and highlights.
- Body and headline font: 'Inter' for a modern and readable UI.
- Use crisp, minimalist icons in white or light gray to ensure legibility against the dark background.
- Employ a card-based layout for titles, making it easy to browse and manage your lists.
- Subtle transitions and animations for a smooth and engaging user experience.
